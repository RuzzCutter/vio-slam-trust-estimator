## Гипотеза

Адаптивная оценка достоверности визуальных измерений снижает навигационную ошибку
и повышает устойчивость локализации по сравнению с классическим VIO при деградации
визуального потока.

---

## Принятые решения

1. **Сравнение:** только OpenVINS baseline vs OpenVINS+adaptive (VINS-Mono/ORB-SLAM3 — литобзор).
2. **Карта достоверности:** входит в первую публикацию, реализуется после адаптивной ковариации.
3. **Данные:** EuRoC + TUM-VI; натурные эксперименты — только описание в тексте.
4. **Диск:** ~57 ГБ свободно; minimal EuRoC скачан (MH_01, MH_04, MH_05).
5. **Срок:** упрощать протокол; часть результатов допустимо экстраполировать.

## 2. Архитектура (два подэтапа)

### 2a. Адаптивная ковариация (сейчас)

```
Датасет → OpenVINS baseline (R = const)
       → OpenVINS adaptive: f₁…f₄ → c(t) → R_t = R₀/(c(t)+ε)
```

Точка интеграции: `ov_msckf/src/update/UpdaterMSCKF.cpp` (`sigma_pix_sq`).

### 2b. Карта достоверности (следующий шаг)

```
c(t) + поза → накопление в сетке/облаке → карта C(x,y,z)
           → штраф в планировщике (раздел 4.4, упрощённо для статьи)
```

Для ускорения: карта на 2D-сетке по позе камеры, без полноценного планировщика —
достаточно показать корреляцию «маршрут через зоны низкого c(t)» vs ATE.

## 3. Модуль trust_estimator

### Признаки (этап без нейросети)

| Признак | Формула |
|---------|---------|
| f₁ | N_inliers / N_features |
| f₂ | exp(−e_repr / 2.0) |
| f₃ | доля заполненных ячеек сетки 8×8 |
| f₄ | exp(−tr(P_pose) / τ) |

```
c(t) = clip(0.25·(f₁+f₂+f₃+f₄), 0.05, 1.0)
R_t = R₀ / (c(t) + 0.05),  cap: R_t ≤ 100·R₀
```

При c(t) < 0.15 — пропуск визуального обновления.

**Реализация:** `ov_msckf/src/trust/TrustEstimator.{h,cpp}`  
**Интеграция:** `UpdaterMSCKF.cpp` — масштабирование `sigma_pix_sq`, лог CSV  
**Конфиг baseline:** `config/euroc_mav` (`trust_enable: false`)  
**Конфиг adaptive:** `config/euroc_mav_trust` (`trust_enable: true`)

```bash
# baseline (без trust)
bash scripts/run_euroc_baseline.sh MH_01_easy --duration 60

# adaptive (trust_estimator)
bash scripts/run_euroc_adaptive.sh MH_01_easy --duration 60
# trust log: datasets/results/<SEQ>_adaptive_*/trust_log.csv
```

---

## 7. Критерий подтверждения гипотезы

1. ATE(adaptive) < ATE(baseline) на ≥ 2 сценариях деградации (разница > 10%).
2. failures(adaptive) ≤ failures(baseline).
3. На чистых данных: ATE(adaptive) ≤ 1.05 × ATE(baseline).
4. mean c(t) падает при усилении деградации.

## 9. Известные проблемы

| Проблема | Решение |
|----------|---------|
| Jazzy: `image_transport.h` not found | локальный патч `__has_include` в ROS2Visualizer.h |
| Jazzy: `SetParameter` import | `from launch_ros.actions import SetParameter` |
| `tcsetattr errno=4` при `--duration` | `--disable-keyboard-controls` в bag play |
| OpenVINS crash `save_total_state` | `save_total_state:=false` в launch |
| MH_01 init failed | `--start-offset 40` (авто в скрипте) |
| `--viz` без bag | bag автозапускается после старта OpenVINS |
| Shutdown warnings в run_simulation | безвредно (ROS 2 Jazzy) |
